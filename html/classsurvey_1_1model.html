template<typename T>
survey::model class

MongoDB data model base class.

Contents

  • Reference

Typical usage:

class pants : public survey::model<pants>
{
public:
    COLLECTION(pants)

    pants() : survey::model<pants>{"test"} {}
};

int main()
{
    mongocxx::instance instance{};
    auto document = pants::get("5b63f486be9ca51a9a3c0e81");

    // ...

    return 0;
}

template<typename T>
void survey::model<T>::save()

Persist the document to the database. This method can be used to:

  • update an existing document; or to
  • create a new document.
// Update a document
auto document = pants::get("5b63f486be9ca51a9a3c0e81");
document.set_data("{\"key\":\"val\"}");
document.save();

// Create a new document
pants document{};
document.set_data("{\"key\":\"val\"}");
document.save();
